🌐 GitHub Pages 배포 (무료 · 추천)
1
GitHub 계정 만들기
github.com 접속 → Sign up → 무료 가입
2
새 저장소 만들기
New repository → 이름 입력 → Add a README file 체크 → Create repository
3
두 파일 업로드
저장소 페이지 → Add file → Upload files
수강신청_학부모.html과 관리자.html 두 파일을 같이 올리세요
4
Pages 활성화
Settings → Pages → Branch: main → Save
잠시 후 https://내계정.github.io/저장소명/ URL 생성 완료!
5
카카오톡으로 공유
https://내계정.github.io/저장소명/수강신청_학부모.html
이 URL을 학부모 단톡방에 공유하거나, 학부모 페이지의 카카오톡 공유 버튼으로 전송하세요.
💡 핵심: 두 파일을 같은 저장소(동일 도메인)에 올려야 localStorage가 공유됩니다.
관리자 페이지에서 수정 → 저장하면 학부모 페이지에 즉시 반영됩니다.
🟡 카카오톡 공유 카드 설정 (선택)
앱 키 없이도 링크 복사로 공유됩니다. 카카오 공유 카드(미리보기 이미지 포함)를 원하시면:

1. developers.kakao.com → 내 애플리케이션 → 앱 추가
2. 앱 키 → JavaScript 키 복사
3. 수강신청_학부모.html 상단의 const KAKAO_KEY = '' 에 붙여넣기
4. 앱 설정 → 플랫폼 → Web → 사이트 도메인에 GitHub Pages 주소 등록
